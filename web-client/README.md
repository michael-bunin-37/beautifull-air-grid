## Project Overview

A high-performance Airtable-like web application for working with large datasets (50k+ rows).  
Built with React, TypeScript, TanStack Table, Node.js (BFF), and PostgreSQL.

Features:

- Virtualized table rendering for large data volumes
- Real-time collaboration across multiple clients
- Inline cell editing (text, numbers, select)
- Optimistic UI updates
- Docker Compose setup for full environment (frontend, backend, database)

Focused on scalability, clean architecture, and smooth user experience.
